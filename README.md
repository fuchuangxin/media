### 系统介绍
    自媒体内容监控平台，通过智能采集微博、微信、政府网站、深圳证券公告等内容源，实现根据采集内容进行关键词筛选、监控、并可进行二次编辑，实现自动推送和一键发布到腾讯内容开放等内容平台。实现了智能采集、智能推送，大幅度节约了编辑等人力资源成本，提高内容分发效率。
